## Чем могу быть полезен

### 🔄 Миграция на Rust / Zig
Ваш сервис на Python, Node.js, Go или Java — перепишу критичные участки на Rust или Zig. Результат:
- Скорость выше в 10–50 раз
- Потребление памяти меньше в 5–10 раз
- Без дампов памяти, без segfault'ов
- Надёжность под высокой нагрузкой

### 🏗 Разработка с нуля
- REST / GraphQL API на Rust (Axum 0.7)
- WebSocket серверы, real-time чаты
- Очереди, фоновые воркеры (Redis, RabbitMQ)
- Telegram / VK / OK Mini Apps
- PWA (Vue 3) — работает как приложение на телефоне

### 🔌 Интеграции
- **1С** — двусторонняя синхронизация номенклатуры, документов
- **Bitrix24** — вебхуки, сделки, контакты, активности
- **ФНС (самозанятые)** — автоматические чеки в день, библиотека lknpd-nalog-api-ts
- **VK API / OK API / Telegram Bot API** — Mini Apps, парсинг групп, боты

### 🛡 Аудит и безопасность
- Проверка кода на уязвимости
- Переписывание опасных мест на Rust (borrow checker гарантирует безопасность памяти)
- Настройка HTTPS, rate limiting, авторизации, шифрования

### 🧹 Оптимизация
- Профилирование и ускорение медленных мест
- Уменьшение расхода памяти
- Рефакторинг legacy-кода

### Zig Audio Service
Subprocess для распознавания и синтеза речи. Rust gateway управляет Zig-процессом через Redis-очередь.

**Особенности:**

1. **Техническое задание** — что должен делать сервис, какие нагрузки
